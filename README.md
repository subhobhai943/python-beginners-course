## ğŸ’» Setting Up Your Environment

### Install Python
1. Go to [python.org](https://python.org)
2. Download Python 3.x (latest version)
3. Run the installer
4. **Important**: Check "Add Python to PATH" during installation

### Choose a Code Editor
We recommend:
- **VS Code** (Beginner-friendly, free) - [Download here](https://code.visualstudio.com/)
- **PyCharm Community** (Feature-rich, free) - [Download here](https://www.jetbrains.com/pycharm/)
- **IDLE** (Comes with Python, simple)

### Verify Installation
Open your terminal/command prompt and type:
```bash
python --version
```
You should see something like "Python 3.x.x"
