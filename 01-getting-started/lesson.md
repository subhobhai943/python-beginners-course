## ğŸ’» Your First Python Program

Let's write the most famous program in programming history - "Hello, World!"

```python
print("Hello, World!")
```

That's it! When you run this, Python will display: `Hello, World!`

### How It Works

- `print()` is a **function** that displays text on the screen
- The text inside quotes `"Hello, World!"` is what gets displayed
- The quotes tell Python that this is text (we call this a "string")

## ğŸ® Interactive Python Shell

Python has an interactive mode where you can type commands and see results immediately!

1. Open your terminal/command prompt
2. Type `python` or `python3` and press Enter
3. You'll see `>>>` - this means Python is ready!
4. Try typing: `print("Hello!")` and press Enter
5. To exit, type `exit()` or press Ctrl+D (Ctrl+Z on Windows)
